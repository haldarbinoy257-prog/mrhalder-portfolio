## ⚙️ Run Locally

Clone the repo:

```
git clone https://github.com/haldarbinoy257-prog/mrhalder-portfolio.git
cd mrhalder-portfolio
```

Install dependencies:

```
pip install -r requirements.txt
```

Run server:

```
python app.py
```

Open:

```
http://127.0.0.1:5000
```
